# Oracle SQL Notes

## Overview

This repository contains structured Oracle SQL notes prepared during my database learning journey. The content is organized topic-wise with explanations, syntax, examples, and practice queries to build a strong foundation in SQL and relational database concepts.

The repository is designed as a quick reference for students, freshers, and aspiring Java Full Stack Developers who want to learn Oracle SQL in a systematic way.

* Introduction to Oracle SQL
* Data Types
* Retrieving Data (SELECT Statement)
* Case Sensitivity
* Constraints
* Tables
* Operators
* Order By Clause
* Functions
* Group By Clause
* Having Clause
* Sub Queries
* Joins
* SQL Commands (DDL, DML, DCL, TCL, DQL)
* Miscellaneous Concepts
* Database Objects
* Difference Between Oracle and MySQL
* Database Clients
* ER Diagram
* ER Schema
* Normalization
* Flat File Model
* Relational Model
* Assignment Queries
