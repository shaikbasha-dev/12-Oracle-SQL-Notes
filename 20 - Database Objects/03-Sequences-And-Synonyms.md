# DATABASE OBJECTS

# 1. Sequence

## Definition

* A **Sequence** is a database object used to generate numbers automatically.
* It is commonly used to generate unique values for primary key columns.
* Sequences generate numbers according to:

  * Starting value
  * Increment value

---

## Creating a Sequence

### Syntax

```sql
CREATE SEQUENCE sequence_name
START WITH start_value
INCREMENT BY increment_value;
```

### Example

#### Question

Create a sequence named **Dec** that starts with 1 and increments by 1.

### Query

```sql
CREATE SEQUENCE Dec
START WITH 1
INCREMENT BY 1;
```

### Output

| Result           |
| ---------------- |
| Sequence Created |

---

## Sequence Keywords

### 1. NEXTVAL

* `NEXTVAL` generates the next value in the sequence.

### Syntax

```sql
sequence_name.NEXTVAL
```

---

### 2. CURRVAL

* `CURRVAL` returns the current value generated by the sequence.

### Syntax

```sql
sequence_name.CURRVAL
```

---

## Using Sequence with INSERT

### Question

Insert records into DEMO table using sequence values.

### Query

```sql
INSERT INTO demo
VALUES(Dec.NEXTVAL, 'Sachin');

INSERT INTO demo
VALUES(Dec.NEXTVAL, 'Virat');
```

### Output

| ID | NAME   |
| -- | ------ |
| 1  | Sachin |
| 2  | Virat  |

---

## Deleting a Sequence

### Syntax

```sql
DROP SEQUENCE sequence_name;
```

### Example

```sql
DROP SEQUENCE Dec;
```

### Output

| Result           |
| ---------------- |
| Sequence Dropped |

---

# 2. Synonym

## Definition

* A **Synonym** is an alternative name given to a database object.
* It is generally used to provide another name for a table.
* Synonyms simplify SQL queries and improve readability.

---

## Creating a Synonym

### Syntax

```sql
CREATE SYNONYM synonym_name
FOR table_name;
```

### Example

#### Question

Create a synonym named **emp_details** for EMP table.

### Query

```sql
CREATE SYNONYM emp_details
FOR emp;
```

### Output

| Result          |
| --------------- |
| Synonym Created |

---

## Using Synonym

### Query

```sql
SELECT *
FROM emp_details;
```

### Output

| Result                              |
| ----------------------------------- |
| Displays all records from EMP table |

---

## Deleting a Synonym

### Syntax

```sql
DROP SYNONYM synonym_name;
```

### Example

```sql
DROP SYNONYM emp_details;
```

### Output

| Result          |
| --------------- |
| Synonym Dropped |

---

# Summary

| Database Object | Purpose                                  |
| --------------- | ---------------------------------------- |
| Sequence        | Generates automatic numbers              |
| NEXTVAL         | Generates next sequence value            |
| CURRVAL         | Returns current sequence value           |
| Synonym         | Provides an alternative name for a table |
